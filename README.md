## How can I ensure that my code, program, or software is functional and secure?
I can ensure that it is functional by testing it, and making sure the test pass, this means testing it for the happy path as well as testing it for errors and exceptions

## How do I approach designing software?
I first create a BRD, then I start creating user stories, then we can make UML diagrams, and choose user stories to begin writing test driven development. The architecture has to be planned as well.
